# AppFusion
> An open-source App Acquisition Intelligence Toolkit

AppFusion mirrors the kind of internal tooling a company like 
Bending Spoons uses post-acquisition — pulling App Store data, 
running AI-powered review analysis, and generating a 90-day 
audit for any app.

## Modules
- **Module 1** — App Store data scraper (iTunes API + MySQL)
- **Module 2** — Retention & monetisation analyser (Pandas + Plotly)
- **Module 3** — AI review intelligence (Google Gemini)
- **Module 4** — Acquisition audit dashboard (React + FastAPI)

## Stack
Python · FastAPI · MySQL · React · Tailwind · Google Gemini API
